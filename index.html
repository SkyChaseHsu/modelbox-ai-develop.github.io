安装指南
Docker安装ModelBox开发镜像
准备工作：确认Ubuntu版本>=16.04；确认已安装Docker
1.获取ModelBox开发镜像
使用SSH工具登陆服务器，自行选择路径，执行以上Docker安装命令行下载ModelBox开发镜像
2.创建启动脚本
在服务器中自行选择路径，创建Docker启动脚本，或在本地将以上脚本按需修改后，粘贴到SSH终端中执行
3.执行命令进入ModelBox
执行命令查看本服务器已启动的所有镜像，保存新启动的镜像ID
docker ps -a | grep modelbox
执行以下命令进入ModelBox
docker exec -it $docker_id bash # 其中$docker_id 指新启动的镜像ID
更多详情请进入 ModelBox快速入门

ModelBox
ModelBox是一套统一的端边云AI应用开发和运行框架的规范，以及在此规范上所实现的运行时框架。在此基础上，各参与方可贡献相关的代码、工具、平台、文档等相关内容
